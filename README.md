# 🍽️ Raízes do Nordeste API

API REST desenvolvida em Node.js para gerenciamento de um restaurante.

## 🚀 Tecnologias

- Node.js
- Express
- MySQL
- JWT (JSON Web Token)
- MVC
- REST API

## 📦 Funcionalidades

- 🔐 Autenticação com JWT
- 👤 Gerenciamento de usuários
- 🍽️ Gerenciamento de produtos
- 📦 Controle de estoque
- 🛒 Gerenciamento de pedidos

## 📁 Estrutura do Projeto

```
src/
├── config/
├── controllers/
├── middlewares/
├── models/
├── routes/
├── services/
```

## ▶️ Como executar

### Clone o projeto

```bash
git clone https://github.com/khauan017/raizes-do-nordeste-api.git
```

### Instale as dependências

```bash
npm install
```

### Configure o arquivo .env

Exemplo:

```env
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=sua_senha
DB_NAME=raizes_do_nordeste
JWT_SECRET=sua_chave_secreta
```

### Execute o projeto

```bash
npm run dev
```
